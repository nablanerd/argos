# Argonautes

## Description

Ajout et Liste [d'argonautes](https://fr.wikipedia.org/wiki/Argonautes) en javascript pour un test technique.

## Stack

- Javascript
- React
- Axios
- Express
- Sequelize

- Cloner le dépôt

## Pour lancer le code

- npm install
- npx sequelize-cli db:migrate
- npx sequelize-cli db:seed:all
- npm start
- cd client;npm install;npm start
